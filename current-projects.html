String Actuated Curved Folded Surfaces

This technology introduces string actuated curved folded surfaces that can be shaped by pulling a network of strings thus vastly simplifying the process of creating such surfaces and making the folding motion an integral part of the design. It solves the problem of which surface points to string together and how to actuate them by locally expressing a desired folding path

Reflectance and Natural Illumination from Single-Material Specular Objects Using Deep Learning

UCL students have developed new deep learning architectures that for the first time achieve sparse data interpolation, mapping from the image to the directional domain, and inferring high dynamic range data from low dynamic range input.

Learn More

Efficient Hybrid Image Warping for High Frame-Rate Stereoscopic Rendering

Anthony Stee and his team evaluate an algorithm with a variety of challenging test cases which show better quality image-warping than state-of-the-art techniques and that it can support transparent surfaces effectively. This theory can achieve image warping at rates suitable for practical use in a variety of applications on modern virtual reality equipment.

Learn More

Perceptual real-time 2D-to-3D conversion using cue fusion

Tobias Ritschel and his team invented a method which is modular and based on priors learned in a pre-process combined with stereo cues extracted from 2D images or videos at runtime.In this way, transfer from 2D to 3D images is more efficient.

Practical Capture and Reproduction of Phosphorescent Appearance

UCL's students design a framework for comprehensive pipeline for digital acquisition, reproduction and manipulation of phosphorescent appearance.
