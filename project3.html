Efficient Hybrid Image Warping for High Frame-Rate Stereoscopic Rendering Research Item

  1. Home
  2. Research
  3. Research Item


Project Description

The research has the aim of improving pervious approaches of warping images using rendering engines for virtual reality, this results in image sizes which aren't well scaled. Furthermore, previous rendering engines attempted to reuse previous frames or rendering lower cost stereo pair due to high cost of rendering fee. However, the actual method to produce well scaled image is running the full render pipeline again. Hence, a new inage-warping algorithm needs to be introduced.

Authors

  • Andre Schollmeyer
  • Simon Schneegans
  • Stephan Beck
  • Anthony Steed
  • Bernd Froehlich

Impact

Thanks to the new algoirthm, it provides novel contributions such as grid generation algorithm for proxy geometry image warping, so as addressing uncclusion with low-pass filling algorithm. The researchers evaluated a new image-warping algorithm that achieves better quality image warping which support transparent surfaces effectively.
